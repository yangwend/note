## 移动端实现预览pdf和下载pdf

项目中经常需要实现在线展示 PDF 文件以及下载 PDF 文件的功能。由于 PC 端和 H5 端不同浏览器有兼容问题，因此考虑以下几种方式进行实现。

### PDFObject
PDFObject 是一个用于在 HTML 中动态嵌入 PDF 文档的 javascript 库。PC 端兼容性比较好，可以兼容 ie9，但是移动端兼容性比较差。
